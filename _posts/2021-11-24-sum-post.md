## 함수의 인자로 시작과 끝 숫자를 받아 시작부터 끝까지의 모든 정수값의 합을 리턴하는 함수를 작성하기
    

#### def sum 함수 만들기
 <pre><code>
def sum(start, end):         
    sum=0                   << sum의 기본값을 0으로 설정
    for a in range(start, end+1):
        sum = sum + a               << for 반복문을 통해 start 부터 end까지 차례로 더한다
    return sum
    
</code></pre>
